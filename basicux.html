DESIGN SPRITE + WEB PROTOTYPE

In the last theme, I analyzed and worked with data about students whos using our school platform Fronter. I have done some user tests and interviews with students from different semesters.

I created this project in Group 4 with Arnar Logi Hakonarson, Janina-Stefania Olteanu, Konstantinos Anagnostou and Petra Gergely. We decided to add to Fronter new function for upload files, folders, documents and share it with other students. Something similar to Google Drive.

Each member of our group created one paper prototype, after that each of us created his own web prototype in Adobe XD. Our group chooses my prototype, so we continued with testing. After testing I changed some things in the prototype and create a coded web prototype.

DESIGN SPRITE: NATIVE APP

The last project of the 1st semester was created to Mobile App for KEA. The requirement was using colors from KEA design guide and UI Kits.

This project was a group project and my group members were Janina-Stefania Olteanu, Konstantinos Anagnostou, Dasa Jarinova, Andrea Makarova, Mateusz Mieciek and Ioana-Andreea Mihai. We created a mobile app like a portfolio for KEA students. Students can upload there their projects, graphic, videos.. everything they want to share with others. This App can be also useful for employers, they can view all projects, find a contact on student or sent them a direct message.

In this theme, I have learned more about Mobile Applications, about UI, and about UX testing. The last assignment in this theme was created pdf with whole 4 weeks documentation.
